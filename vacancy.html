Senior Java Developer (backend development)

Требования:
  • хорошее знание Java или Scala, реляционных БД, структур данных и алгоритмов
  • системный подход к решению задач, логика
  • умение работать с Mercurial или Git
  • желание учиться
Желательно:
  • J2EE
  • функциональное программирование
  • опыт модульного и приемочного тестирования
  • знания о высоконагруженных системах
  • навыки frontend-разработки
  • опыт, которым вы можете поделиться с нами

 

Web Developer

Требования:
  • HTML5, CSS3, javascript на отличном уровне
  • опыт написания и сопровождения большого веб-приложения
Желательно:
  • LESS
  • AngularJS
  • кросс-браузерная верстка
